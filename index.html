Elementary HTML

Creating Lists

  • Unordered List: A list with bullets. 
    <ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>
  • Ordered List: A list with numbers. 
    <ol>
  <li>First</li>
  <li>Second</li>
</ol>
  • Description List: 
    <dl>
  <dt>HTML</dt>
  <dd>HyperText Markup Language</dd>
</dl>
  • Nested List: 
    <ul>
  <li>Fruit
    <ul>
      <li>Apple</li>
      <li>Banana</li>
    </ul>
  </li>
</ul>

HTML Table Tags

Use tables to organize data.

<table>
  <caption>Student Marks</caption>
  <tr>
    <th rowspan="2">Name</th>
    <th colspan="2">Subjects</th>
  </tr>
  <tr>
    <th>Math</th>
    <th>Science</th>
  </tr>
  <tr>
    <td>Alice</td>
    <td>90</td>
    <td>95</td>
  </tr>
</table>

Formatting Text

Use the <font> tag for text formatting (not recommended in HTML5).

<font color="green" size="4" face="Verdana">Hello, World!</font>

Setting Page Margins

<style>
  body {
    margin: 40px;
  }
</style>

Creating Forms in HTML

<form>
  Name: <input type="text" name="username"><br>
  Message: <textarea name="message"></textarea><br>
  Gender:
  <select name="gender">
    <option value="male">Male</option>
    <option value="female">Female</option>
  </select>
</form>

Tweening and Publishing (Flash)

  • Motion Tweening: Moves an object smoothly from one position to another.
  • Shape Tweening: Morphs one shape into another.
  • Scenes: Break your animation into parts.
  • Publishing: Export as .swf or .html for viewing.
  • Embed Flash: 
    <embed src="animation.swf" width="550" height="400">

World Wide Web

  • Protocols: HTTP, HTTPS, FTP.
  • Components: Web browsers, web servers, URLs.
  • Web Hosting: Providing space on a server for websites.
